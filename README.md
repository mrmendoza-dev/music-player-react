# React Music Player
Music player with general media controls. Plays local music files and displays info about the song and shows progess live.

### [Live Demo](https://music-player-mendoza.netlify.app/)

### Features
- Play/Pause
- Previous/Next Song
- Repeat Functions (All/Single)
- Shuffle Mode
- Volume Control
  
### Future Updates
- Music Visualizers
- EQ Editors
- Spotify Integration
